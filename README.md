## 📌 Project Overview
This project delivers a deep-dive analysis into consumer habits using a dataset of **3,900 transactions**. By bridging the gap between raw data and business strategy, I utilized a full-stack data pipeline: **Python** for automated cleaning, **MySQL** for relational data exploration, and **Power BI** for executive-level storytelling.
